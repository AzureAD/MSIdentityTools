## SYNOPSIS

Convert Msft Identity token structure to PowerShell object.

## SYNTAX

```powershell
ConvertFrom-MsIdJwtToken [-Tokens] <String[]> [-ProgressAction <ActionPreference>] [<CommonParameters>]
```
