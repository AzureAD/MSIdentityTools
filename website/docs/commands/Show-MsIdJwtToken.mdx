## SYNOPSIS

Show Json Web Token (JWT) decoded in Web Browser using diagnostic web app.

## SYNTAX

```powershell
Show-MsIdJwtToken [-Tokens] <String[]> [-OAuth2RedirectUri <Uri>] [-Force] [-ProgressAction <ActionPreference>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### EXAMPLE 1

```powershell
$MsalToken.IdToken | Show-MsIdJwtToken
```
