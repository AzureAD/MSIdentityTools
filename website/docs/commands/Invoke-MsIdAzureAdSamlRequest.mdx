## SYNOPSIS

Invoke Saml Request on Azure AD.

## SYNTAX

```powershell
Invoke-MsIdAzureAdSamlRequest [-SamlRequest] <Object[]> [-TenantId <String>]
 [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

### EXAMPLE 1

```powershell
$samlRequest = New-MsIdSamlRequest -Issuer 'urn:microsoft:adfs:claimsxray'
PS > Invoke-MsIdAzureAdSamlRequest $samlRequest.OuterXml
```
